Superstore Data Analytics Tool (Python EDA)
Files: Data Analytics Tool.ipynb, superstore.xlsx

Project Overview: Developed a Jupyter-based tool to automate the analysis of retail "Superstore" data.

Key Features:

  Automated EDA: Created functions to generate instant summary statistics and distribution plots.

  Trend Discovery: Identified seasonal sales patterns and underperforming regions.

  Profitability Analysis: Deep dive into discount impacts on overall margins.
